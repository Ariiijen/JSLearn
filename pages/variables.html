Declaring Variables

Use the var keyword to declare a variable. You can declare with or without an initial value.

Syntax
var variableName; var variableName = value;
Example
var status; // declared, no value yet var age = 20; // number value var name = "Alice"; // string value var isStudent = true; // boolean value alert(name + " is " + age); // "Alice is 20"

Variable Scope

Scope determines where a variable can be accessed in your code.

Global vs Local
var greet = "Hello "; // GLOBAL ‚Äî accessible anywhere function sayHello() { var name = "Vince!"; // LOCAL ‚Äî only inside this function alert(greet + name); // ‚úÖ works: "Hello Vince!" } sayHello(); alert(greet); // ‚úÖ works: "Hello " alert(name); // ‚ùå ERROR: name is not defined here
TypeWhere declaredAccessible
GlobalOutside any functionAnywhere in the script
LocalInside a functionOnly inside that function

Naming Rules

  • Variable names are case-sensitive ‚Äî name and NAME are different variables
  • Can contain letters, digits, $, and _
  • Cannot start with a digit (e.g. 1score is invalid)
  • Cannot use reserved keywords like var, function, return
Valid vs Invalid Names
// ‚úÖ Valid names var myName; var score1; var _private; var $price; // ‚ùå Invalid names var 1score; // starts with digit var my-name; // hyphens not allowed var function; // reserved keyword
