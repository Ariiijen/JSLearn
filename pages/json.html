What is JSON?

JSON stands for JavaScript Object Notation. It is a lightweight text format designed for human-readable data exchange. Key facts:

  • File extension: .json
  • Media type: application/json
  • Created by Douglas Crockford
  • Used by web APIs to send/receive data
  • Readable by almost every programming language

JSON Syntax Rules

  • Data is in name : value pairs
  • Names (keys) must always be in double quotes
  • Objects are wrapped in { }
  • Arrays are wrapped in [ ]
  • Items are separated by commas
Example JSON Structure
{ "student": { "name": "Alice Johnson", "age": 20, "enrolled": true, "courses": ["Math", "IT", "Science"] } }

JSON.parse()

Converts a JSON string into a JavaScript object. Used when receiving data from a server.

Syntax
var obj = JSON.parse(jsonString);
Example
var jsonStr = '{"name":"Alice","age":20}'; var person = JSON.parse(jsonStr); person.name // "Alice" person.age // 20

JSON.stringify()

Converts a JavaScript object into a JSON string. Used when sending data to a server.

Syntax
var str = JSON.stringify(object);
Example
var student = { name: "Alice", age: 20, enrolled: true }; var json = JSON.stringify(student); // '{"name":"Alice","age":20,"enrolled":true}'
