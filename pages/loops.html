Loops let you repeat a block of code multiple times. JavaScript has for, while, and for...of loops.

On this page
  • for Loop
  • while Loop
  • for...of Loop
  • Try It

for Loop

Best when you know exactly how many times to repeat. Has three parts: initializer, condition, iteration.

Syntax
for (initializer; condition; iteration) { // code to repeat }
Example ‚Äî Count 0 to 4
for (var i = 0; i < 5; i++) { console.log(i); } // Output: 0 1 2 3 4

while Loop

Repeats as long as a condition stays true. You control the counter manually.

Syntax
while (condition) { // code to repeat }
Example ‚Äî Count 0 to 4
var i = 0; while (i < 5) { console.log(i); i++; } // Output: 0 1 2 3 4

for...of Loop

Loops through the elements of an iterable (array, string, map) one by one.

Syntax
for (const item of iterable) { // code for each item }
Example ‚Äî Iterate Array & String
// Array iteration const fruits = ["apple", "mango", "banana"]; for (const fruit of fruits) { console.log(fruit); } // Output: apple mango banana // String iteration const name = "JS"; for (const ch of name) { console.log(ch); } // Output: J S
