What is the DOM?

When a browser loads an HTML page, it creates a tree structure of all elements. JavaScript can navigate and modify this tree to make pages dynamic.

Using the DOM, JavaScript can:

  • Change HTML elements and attributes
  • Change CSS styles
  • Add or remove elements
  • Respond to events (clicks, key presses, etc.)

getElementById()

Selects a single element by its unique id. Returns null if not found.

Syntax
document.getElementById("id-name");
Example
// HTML: <p id="para1">Hello!</p> const para = document.getElementById("para1"); para.textContent = "Text changed by JS!"; para.style.color = "#f0c040";

querySelector()

Selects the first element matching a CSS selector. Use . for class, # for id, or tag name directly.

Syntax
document.querySelector("selector");
Example
document.querySelector("h1"); // first <h1> document.querySelector(".card"); // first element with class="card" document.querySelector("#title"); // element with id="title"

querySelectorAll()

Returns a NodeList (like an array) of ALL elements matching the selector.

Syntax
document.querySelectorAll("selector");
Example
const items = document.querySelectorAll("ul > li"); items.forEach(function(item) { item.style.color = "#3ecf8e"; });

createElement() & appendChild()

Create new HTML elements and add them to the page.

Syntax
var el = document.createElement("tagName"); el.textContent = "text"; parentElement.appendChild(el);
Example
var ul = document.createElement("ul"); var li = document.createElement("li"); li.textContent = "New item added by JS"; ul.appendChild(li); document.body.appendChild(ul);
