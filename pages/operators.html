⚙️ Operators

Operators perform operations on values. JavaScript has arithmetic, comparison, logical, and assignment operators.

On this page
  • Arithmetic
  • Comparison
  • Logical
  • Assignment
  • Try It

Arithmetic Operators

Perform mathematical calculations on numbers.

OperatorNameExampleResult
+Addition5 + 1015
-Subtraction10 - 55
*Multiplication5 * 1050
/Division10 / 52
%Modulus5 % 21
++Incrementx++x + 1
--Decrementx--x - 1
Example
var x = 5, y = 10; x + y // 15 y - x // 5 x * y // 50 y / x // 2 x % 2 // 1 (remainder of 5÷2)

Comparison Operators

Compare two values and return true or false.

OperatorNameExampleResult
==Equal (value)5 == "5"true
===Equal (value + type)5 === "5"false
!=Not equal5 != 10true
>Greater than10 > 5true
<Less than5 < 10true
>=Greater than or equal5 >= 5true

Logical Operators

Combine multiple conditions.

OperatorNameExampleResult
&&AND — both must be true(5 < 10) && (3 > 1)true
||OR — at least one true(5 > 10) || (3 > 1)true
!NOT — reverses value!(5 < 10)false

Assignment Operators

Assign or update a variable's value.

Example
var x = 5; x += 1; // x = 6 (same as x = x + 1) x -= 1; // x = 5 x *= 5; // x = 25 x /= 5; // x = 5 x %= 2; // x = 1 (remainder)
