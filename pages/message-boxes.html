JavaScript provides three built-in popup functions to communicate with users: alert(), confirm(), and prompt().

On this page
  • alert()
  • confirm()
  • prompt()
  • Try It

alert() ‚Äî Simple Message

Displays a message to the user. Has only an OK button. Use it to show important notices.

Syntax
alert(message);
Example
alert("This is an alert box."); alert(42); // numbers work too alert(true); // booleans work too
Note: alert() accepts any data type ‚Äî strings, numbers, booleans ‚Äî no conversion needed.

confirm() ‚Äî Yes or No

Shows a message with OK and Cancel buttons. Returns true if OK is clicked, false if Cancel.

Syntax
var result = confirm(message);
Example
var choice; if (confirm("Do you want to save changes?") == true) { choice = "Data saved!"; } else { choice = "Save cancelled."; } alert(choice);

prompt() ‚Äî Ask for Input

Displays a text input box. The first parameter is the message; the second is the default value shown in the input field. Returns the typed value, or null if cancelled.

Syntax
prompt(message, defaultValue);
Example
var age = prompt("Please enter your age", "18"); if (age != null) { alert("You are " + age + " years old"); }
