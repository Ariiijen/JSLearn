Use if, if-else, and else if to control which code runs based on conditions.

On this page

  • Try It

if Statement

Runs a block of code only when the condition is true.

Syntax
if (condition) { // code runs if condition is true }
Example
var score = 85; if (score >= 75) { alert("You passed!"); }

if-else Statement

Runs one block if the condition is true, and a different block if it's false.

Syntax
if (condition) { // runs when true } else { // runs when false }
Example
var score = 60; if (score >= 75) { alert("You passed!"); } else { alert("You failed. Try again."); }

else if Statement

Check multiple conditions in sequence. The first true condition runs its block.

Syntax
if (condition1) { // runs if condition1 is true } else if (condition2) { // runs if condition2 is true } else { // runs if none are true }
Example ‚Äî Grade Checker
var score = 88; if (score >= 90) { alert("Grade: A"); } else if (score >= 80) { alert("Grade: B"); } else if (score >= 75) { alert("Grade: C"); } else { alert("Grade: F ‚Äî Below passing"); }
