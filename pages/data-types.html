String

A sequence of characters enclosed in single or double quotes. You can access individual characters by index (starts at 0).

Syntax
var name = "Hello World";
Example
var str = 'Hello World'; str[0] // "H" str[6] // "W" str.length // 11

Number

JS has one number type that covers integers, decimals, hex, binary, and more.

Syntax
var num = 100;
Example
var a = 100; // integer var b = -100; // negative var c = 10.52; // decimal (float) var d = 0xfff; // hexadecimal var e = 256e-5; // exponential

Boolean

Only two values: true or false. Commonly used in conditions.

Syntax
var flag = true;
Example
var isLoggedIn = true; var hasError = false; if (isLoggedIn) { alert("Welcome back!"); }

Array

An ordered list of values stored in a single variable. Items are accessed by index starting at 0.

Syntax
var arr = [value1, value2, value3];
Example
var fruits = ["apple", "banana", "mango"]; var nums = [1, 2, 3, 4]; var mixed = [1, "two", true]; fruits[0] // "apple" fruits.length // 3

Object

A collection of key-value pairs (properties). Defined with curly braces.

Syntax
var obj = { key: value, key2: value2 };
Example
var student = { name: "Alice", age: 20, course: "IT" }; student.name // "Alice" student.age // 20

Null & Undefined

Null ‚Äî intentionally empty. Undefined ‚Äî declared but no value assigned yet.

Example
var x = null; alert(x); // null var y; alert(y); // undefined
