JavaScript in <head>

You can place a <script> tag inside the <head> section. The script runs before the page body is loaded.

Syntax
<head> <script> // your JavaScript here </script> </head>
Example
<head> <script> alert("Hello from the head!"); </script> </head> <body> <h2>JavaScript in Head</h2> </body>

JavaScript in <body>

Placing <script> at the end of <body> is often preferred ‚Äî the HTML loads first, then the script runs.

Syntax
<body> <!-- HTML content --> <script> // your JavaScript here </script> </body>
Example
<body> <h2>My Page</h2> <script> alert("Hello from the body!"); </script> </body>

External JavaScript File

Save your JS in a separate .js file and link it with the src attribute. This is the most common real-world approach.

Syntax
<script src="filename.js"></script>
3 Ways to Reference
// Full URL <script src="https://example.com/js/script.js"></script> // File path <script src="/js/script.js"></script> // Same folder <script src="script.js"></script>
‚ö† Warning: External script files cannot contain <script> tags inside them ‚Äî only plain JavaScript code.

Advantages of External JS

  • Separates HTML and JavaScript code ‚Äî cleaner files
  • Easier to read, maintain, and debug
  • One file can be reused across many pages
  • Browsers cache the JS file ‚Äî speeds up page loads
Tip: When adding multiple JS files to one page, just add multiple <script> tags one after another.
